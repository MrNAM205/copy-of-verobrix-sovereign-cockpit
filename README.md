# Verobrix Sovereign Cockpit

A unified sovereign cockpit for lawful knowledge, AI cognition, and remedy generation. Features a comprehensive legal corpus, intelligent document drafting, and immutable archival of sovereign acts.

## Features

*   **Lawful Knowledge & AI Cognition**
*   **Remedy Generation & Intelligent Drafting**
*   **Immutable Archival of Sovereign Acts**

## Tech Stack

*   **Framework:** React 19, Vite
*   **Language:** TypeScript
*   **AI:** Google GenAI SDK

## Project Links

*   [View your app in AI Studio](https://ai.studio/apps/drive/1Jh6cYhmIxHQLkZ1HdT2N28PfckB_7K57)

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
